## 説明文（App Store 掲載用）

TOEIC L&R テストの解答を記録・採点できるシンプルなアプリです。

【主な機能】
- 解答シートの作成と管理
- ユーザー回答と正解の入力
- 自動採点とスコア算出（リスニング・リーディング別）
- パート別正答率のグラフ表示
- 間違えた問題の一覧表示
- 学習履歴の記録と振り返り

【使い方】
1. 新しい解答シートを作成
2. テスト後に自分の回答を入力
3. 正解を入力して自動採点
4. パート別の正答率や弱点を確認

【特徴】
- オフラインで完結 — ネット接続不要
- シンプルな操作 — 直感的な UI で素早く入力
- プライバシー重視 — データはすべてデバイス内に保存
- 広告なし — 学習に集中できる環境

TOEIC のスコアアップを目指す方の日々の学習をサポートします。
